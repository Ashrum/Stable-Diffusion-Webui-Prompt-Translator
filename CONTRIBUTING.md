# Contributing to Stable-Diffusion-Webui-Prompt-Translator

## Code of Conduct

Please be respectful in issues and pull requests. Harassment, discrimination, or abusive language will not be tolerated.

## How to Contribute

1. **Open an issue first.** Describe the bug, feature, or translation issue before submitting a PR.
2. **One change per pull request.** Keep PRs small and focused.
3. **Preserve behavior.** This extension runs inside Stable Diffusion WebUI. Preserve compatibility unless the scope of a PR explicitly covers an integration change.
4. **Translations and docs.** Contributions to README, docs/, or language files are welcome.

## Development

- `javascript/prompt_translator.js` — frontend translation UI and toolbar behavior
- `scripts/prompt_translator.py` — backend translation API routing

## Maintainer Notes

The maintainer reviews PRs on a rolling basis. For larger changes, expect a discussion in the issue queue before implementation.
